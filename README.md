This is a lightweight and user-friendly web application developed using PHP and MySQL to manage rental properties. The system allows property owners to manage their rooms and shops efficiently. Whether you are managing residential rooms or commercial shops, this platform simplifies the process of tracking availability, handling bookings, and managing tenant information.
Key Features
Property Management: Easily add, update, and remove rooms or shops from the system.

User-Friendly Dashboard: A clean interface to manage all rental activities in one place.

Database Integration: Securely stores property and tenant data using MySQL.

Responsive Design: Built with simple CSS to ensure it looks good on different devices.

Easy Deployment: Simple file structure, making it easy to set up on any local server like XAMPP or WAMP.

Technologies Used
Backend: PHP

Database: MySQL

Frontend: HTML5, CSS3
